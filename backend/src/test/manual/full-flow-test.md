# 全流程测试文档

## 测试目标

验证材料平台的完整业务流程，包括用户登录、资料上传、积分管理、搜索、下载、提问、评论、点赞和热搜展示等核心功能。

## 测试环境

- 前端：Vue 3 + Element Plus
- 后端：Spring Boot 2.7.15 + MyBatis-Plus
- 数据库：MySQL 8.0+
- 服务地址：http://localhost:8082/api

## 测试步骤

### 1. 用户登录

**测试步骤：**
1. 访问登录页面
2. 输入学号：2025241887
3. 输入密码：123456
4. 点击登录按钮

**预期结果：**
- 登录成功，跳转至首页
- 前端存储token和用户信息到localStorage
- 后端返回用户信息和token

### 2. 上传资料

**测试步骤：**
1. 点击"上传资料"按钮
2. 填写资料名称：测试资料
3. 填写资料描述：这是一份测试资料
4. 选择分类：工作资料
5. 输入标签：测试,示例
6. 选择一个本地文件（如PDF、Word文档等）
7. 点击"上传"按钮

**预期结果：**
- 上传成功，显示成功提示
- 后端返回上传成功信息
- 用户积分增加20分
- 上传的资料显示在个人中心的上传历史中

### 3. 积分增加

**测试步骤：**
1. 上传资料成功后，点击"个人中心"
2. 查看用户积分

**预期结果：**
- 积分显示为20分（初始为0，上传资料获得20分）
- 上传资料数显示为1

### 4. 搜索资料

**测试步骤：**
1. 在首页搜索框输入"测试"
2. 点击搜索按钮

**预期结果：**
- 搜索结果页面显示包含"测试"关键词的资料
- 后端记录搜索日志
- 搜索关键词"测试"出现在搜索历史中

### 5. 下载扣积分

**测试步骤：**
1. 在搜索结果中选择一份需要积分的资料
2. 点击"下载"按钮
3. 确认下载

**预期结果：**
- 下载成功，文件开始下载
- 用户积分扣除相应的积分（如资料需要5分，则积分变为15分）
- 下载资料数显示为1
- 下载记录显示在个人中心的下载历史中

### 6. 提问

**测试步骤：**
1. 点击"提问"按钮
2. 填写问题标题：如何使用测试资料？
3. 填写问题内容：我想了解如何有效使用这份测试资料
4. 选择分类：学习资料
5. 点击"提交"按钮

**预期结果：**
- 提问成功，显示成功提示
- 问题显示在问题列表中
- 后端记录问题信息

### 7. 评论

**测试步骤：**
1. 进入问题详情页面
2. 在评论框输入：这份资料很有用，谢谢分享！
3. 点击"评论"按钮

**预期结果：**
- 评论成功，显示成功提示
- 评论显示在问题详情页的评论列表中
- 后端记录评论信息

### 8. 点赞

**测试步骤：**
1. 在资料详情页找到点赞按钮
2. 点击点赞按钮

**预期结果：**
- 点赞成功，点赞数增加1
- 后端记录点赞信息
- 按钮状态变为已点赞

### 9. 热搜展示

**测试步骤：**
1. 多次搜索同一个关键词（如"测试"）
2. 查看首页的热搜排行榜

**预期结果：**
- 热搜排行榜显示热门搜索关键词
- "测试"关键词出现在热搜榜中
- 热搜榜按搜索次数排序

## 测试注意事项

1. 确保网络连接正常
2. 确保后端服务和数据库正常运行
3. 测试文件大小不要超过100MB
4. 测试过程中注意观察控制台是否有错误信息
5. 测试完成后清理测试数据
